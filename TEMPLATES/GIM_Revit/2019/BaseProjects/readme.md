Базовые шаблоны файла проекта Revit

Это шаблоны, на основе которых собираются шаблоны отдельных разделов.
Например, базовый шаблон АР порождает шаблоны для разделов АС, АИ, МОДИ и т.д. - т.е. эти шаблоны являются дочерними по отношению к базовому
Эти шаблоны обновляются BIM-специальствами первыми и уже в последствии обновляются дочерние шаблоны и свяанные с ними префабы
